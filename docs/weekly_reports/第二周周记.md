## 前端部分

这周大部分时间都花在了 **Vue 工程化** 上，这也是我第一次认认真真做一个前后端分离的项目。一开始上手的时候确实有点懵，面对堆积如山的原有代码，完全不知道从哪下手。后来跟着课程一步步摸索，慢慢把那些零散的前端代码抽成了**选项式 API**，再到后面学着用**组合式 API**，把能复用的组件都单独抽了出来。  

另外还把所有的 API 接口统一管理起来，接口路径整理到单独的文件夹里，后续维护起来一目了然。

不过也因为前端代码的拆分和重构花了不少时间，这周的代码提交比平时晚了一些。虽然时间上有点赶，但看着代码结构越来越清晰，心里还是挺有成就感的。

---

## 后端部分

后端这边业务上的改动不大，主要是学了一些新东西，也试着落地了一部分。

先是接触了 **MyBatis Plus**，不过暂时还没用到项目里（其实是来不及改了，哈哈），先用 **DTO** 简化了三层架构之间来回传数据的冗余代码，清爽了不少。  

接着重点学了 **面向接口编程**，还配置了**全局异常处理器**，终于不用在每个方法里都裹一层异常处理了。

另外写了一个**管理日志的切面**，以 Controller 和 Service 为切入点，把各个方法的运行情况记录下来，同时将日志持久化到文件里。这样后续想统计方法运行时间、排查问题，翻日志就方便多了。

这周还学了 Maven 的**分模块设计、继承和聚合**，不过我的项目结构目前还比较简单，暂时没怎么体现出来。

这周学后端的**面向切面编程（AOP）**给了我挺大的启发，突然发现前端的拦截器其实也是类似的思路，仔细想想还挺有意思的。  

我也开始有意识地去理解 Spring Boot 框架的底层原理，不再只是停留在“会用”的阶段。
